Step 1 - pip install -r requirements.txt

Step 2 - Edit telemon.py and fill the required details: api_id, api_hash, and phone_number.

Step 3 - Populate the channels.txt with required channels in the format provided in the file itself.

Step 4 - Populate the keywords.txt with keywords you want to monitor the channels for; one per line.

Step 5 - Type "python create_db.py" in terminal from the path

--this will create a database file with required tables that will store your monitored data

Step 6 - Type "python telemon.py" in terminal from the path

Step 7 - Enter the OTP received over Telegram to authenticate access.

Step 8 - Open http://127.0.0.1:5000 in your web browser
